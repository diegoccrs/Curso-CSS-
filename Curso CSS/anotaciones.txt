SELECTORES

Selector de ID: Selecciona un único elemento basado en su atributo id. 
Se define con un símbolo de almohadilla, por ejemplo, #titulo seleccionará el elemento con el ID "titulo".

Selector de clase: Selecciona todos los elementos que tienen un atributo class específico. 
Se define con un punto, por ejemplo, .texto seleccionará todos los elementos con la clase "texto".

Selector Universal: Selecciona todo en el documento, se utiliza *
